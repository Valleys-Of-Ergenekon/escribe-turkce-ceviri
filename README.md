# escribe-turkce-ceviri
"Evolv EScribe Suite için Türkçe Çeviri Yaması"


Masaüstündeki EScribe simgesine sağ tıklayıp "Dosya konumunu aç" deyin.

Açılan klasörde Translations klasörünü bulup içine girin.

İndirdiğiniz tr.po ve tr.mo dosyalarını bu klasörün içine yapıştırın.

EScribe programını açın.

Sol üstteki Options > Language menüsüne gelin ve listeden Türkçe'yi seçin.

Ekrana gelen uyarıya "Yes" deyin. Program yeniden başlayacak ve Türkçe olacaktır.
